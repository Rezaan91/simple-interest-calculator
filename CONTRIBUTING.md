# Contributing

All **contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome**.

## How to contribute
- Fork the repo and create a topic branch.
- Make small, focused changes; include tests where relevant.
- Run any checks locally and ensure the script still works.
- Open a Pull Request describing what changed and why.

## Reporting issues
- Use a clear title and steps to reproduce.
- Include OS / shell, script version, and any logs.

## Commit style
- Write descriptive commit messages.
