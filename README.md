# github-final-project — Simple Interest Calculator (Bash)

> **Important:** Per the assignment rubric, this repository **must be named** `github-final-project`.

A tiny Bash utility that computes **simple interest** given principal (P), annual rate (%) (R), and time in years (T).

## Formula
Simple Interest = P × R × T / 100

## Usage
1) Make the script executable:
```bash
chmod +x simple-interest.sh
```

2) Run it and follow prompts:
```bash
./simple-interest.sh
```

## Example
Input:
- P = 10000
- R = 7.5
- T = 2

Output:
- Simple Interest: 1500.00
- Total Amount (Principal + Interest): 11500.00
